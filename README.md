## Usage

### Install

*Install*
```bash
npm install
```

### bundle dev mode (*+ redux-devtools*)

*Install*
```bash
npm install
```

*General case:*
```bash
npm run dev
```
*New Images*
```bash
npm run build
```

*Deploy to Product Steps*
```bash
npm run build
npm run prod:start
git add .
git commit -m "deploy to production"
git push heroku master
```
